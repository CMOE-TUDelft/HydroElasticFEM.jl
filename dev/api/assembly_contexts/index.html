Assembly Contexts

Assembly contexts carry analysis-dependent metadata (domains, frequency, time, stabilisation parameter) and are passed to all weak-form methods.

Types

HydroElasticFEM.AssemblyContexts.AbstractAssemblyContext — Type
AbstractAssemblyContext

Abstract supertype for context objects passed to element and system assembly routines. Contexts carry analysis-dependent metadata such as active domains, time/frequency coordinates, and optional stabilization parameters.

HydroElasticFEM.AssemblyContexts.FrequencyAssemblyContext — Type
FrequencyAssemblyContext(domains, ω, αₕ)

Assembly context for frequency-domain problems.

  • domains: geometric/physical domains involved in the integration.
  • ω: angular frequency.
  • αₕ: optional stabilization parameter (nothing when disabled).
HydroElasticFEM.AssemblyContexts.TimeAssemblyContext — Type
TimeAssemblyContext(domains, t, αₕ)

Assembly context for time-domain problems.

  • domains: geometric/physical domains involved in the integration.
  • t: current physical time.
  • αₕ: optional stabilization parameter (nothing when disabled).

Accessors

HydroElasticFEM.AssemblyContexts.domains — Function
domains(ctx)

Return the domain collection used in integration and stored in an assembly context.

HydroElasticFEM.AssemblyContexts.frequency — Function
frequency(ctx)

Return the angular frequency ω from a frequency-domain context.

HydroElasticFEM.AssemblyContexts.current_time — Function
current_time(ctx)

Return the current physical time t from a time-domain context.

HydroElasticFEM.AssemblyContexts.stabilization_parameter — Function
stabilization_parameter(ctx)

Return the stabilization parameter αₕ stored in the context. This can be nothing when stabilization is disabled.

HydroElasticFEM.AssemblyContexts.has_stabilization — Function
has_stabilization(ctx)

Return true when the context carries a non-nothing stabilization parameter.

HydroElasticFEM.AssemblyContexts.with_time — Function
with_time(ctx, t)

Create a new TimeAssemblyContext from ctx with updated time t, preserving domains and stabilization settings.
